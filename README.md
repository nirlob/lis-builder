# Aplicación GTK 4 Simple

Una aplicación Python simple con GTK 4 que demuestra los componentes básicos.

## Requisitos

- Python 3.7+
- PyGObject
- GTK 4 (necesario para el sistema)

## Instalación

### En macOS con Homebrew:

```bash
brew install gtk4 libadwaita pygobject3
```

### Instalar dependencias de Python:

```bash
pip install -r requirements.txt
```

## Ejecución

```bash
python3 main.py
```

## Características

- Ventana principal con interfaz GTK 4
- Campo de entrada de texto (Entry)
- Label que actualiza dinámicamente
- Botón para saludar
- Botón para salir

## Estructura

- `main.py` - Aplicación principal con la lógica GTK 4
- `requirements.txt` - Dependencias de Python
